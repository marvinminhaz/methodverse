# Methodverse

A fast, lightweight reference engine for Python built-in methods. Get quick explanations and examples without digging through official documentation.

## 🚀 Features
* **Quick Lookups:** Get method definitions and examples instantly.
* **Flexible:** Use it via terminal or import it directly into your own scripts.
* **Modular:** Designed to easily expand into libraries like Pandas and NumPy.

## 📂 Structure
```text
methodverse/
└── python.py  # Core logic and method dictionaries (str, list, dict, etc.)
```

## 🗺 The Vision

Methodverse is currently in its "Python Core" phase. The roadmap includes:

- [ ] **Pandas Module:** Adding DataFrame and Series method documentation.
- [ ] **NumPy Module:** Adding array and math function references.
- [ ] **CLI Polish:** Improving the interface for faster terminal-based lookups.
- [ ] **Library Packaging:** Converting to a standard pip installable package.
